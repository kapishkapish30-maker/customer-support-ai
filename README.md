# 🤖 Customer Support AI
A full-stack AI-powered customer support chatbot built using React, FastAPI, and Google's Gemini AI.

## Features
- 🤖 Multi-Agent AI
- 💳 Billing Support
- 💻 Product Support
- 🔧 Technical Support
- 📢 Complaint Handling
- ❓ FAQ Assistant
- 💬 Modern Chat Interface
- ⚡ FastAPI Backend
- ⚛️ React Frontend
- 🧠 Gemini AI Integration

## Tech Stack
Frontend
- React
- Vite
- Axios

Backend
- FastAPI
- Python
- Gemini API

AI
- Google Gemini 2.5 Flash

## Installation
### Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev
